---
title: "Get Data Recipient Details"
public: true
openapi: "/openapi-outbound.yaml GET /api/v1/recipients/{client_id}"
---

Use the `/api/v1/recipients/{client_id}` endpoint to get additional details for a particular Data Recipient, including the scopes and redirect URLs.

To successfully call this endpoint, you must first call the [/Token](/api/outbound/endpoints/authorize/token) endpoint to obtain a valid `access_token` using the following settings:

- grant_type: `client_credentials`
- client_id: `{partner client_id}`
- client_secret: `{partner client_secret}`
- scope: `client:admin`
